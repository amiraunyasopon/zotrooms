## What it does

ZotRooms uses React to create a user friendly platform to search for available study rooms. This is made possible with Anteater API which fetchs data on the study rooms all over campus. Users are then able to search through available study rooms with specifics such as dates, time slots, and max capacities.

<!-- ![image](https://github.com/user-attachments/assets/397f49e3-6586-4a4d-a4ab-e03247466863) -->
This system is accessed via our frontend, built with React and Next.js.
